Seafile搭建个人网盘

Posted on 2020-01-16 | Post modified: 2020-01-24 | |

Seafile 搭建个人网盘。

（1）背景

目的是创建一个同步盘，在不同的地点、不同的设备上进行同步文件。现有的百度云等服务器不适合作为个人大文件存储，因为有限速、审核等问题。常见的同步盘还有 nextcloud，其优点是简单，但是 seafile 中的文件是以块存储，所以运行效率更高。

（2）seafile 介绍

Seafile 是一个开源的文件云存储平台，主要为团队设计。分为开源免费版和专业版，专业版中多了搜索，集群（亚马逊，阿里云）等功能，并且专业版如果三个用户，那么是免费的，所以个人使用的话，建议使用专业版。如果人多没钱，那么使用开源版。有钱，使用专业版。更多功能对比见这里。

并且Seafile 是开源项目，因此相对数据保密性是有保证的，起码不用担心有什么看不见的后门。

（3）安装实践

安装整体流程：建议基于docker 安装，数据库采用mysql 数据库。如果服务器上有别的程序运行，那么需要设置反向代理 nginx。

国内vps 和国外vps 的选择？

国内vps 速度相对国外的vps 肯定是快的，但是不能科学上网，所以常见的用途：

  • 搭建图床
  • seafile 网盘
  • Jupyterlab服务

国外 vps 流量很便宜，但是硬盘往往比较小，速度还行。主要用途：

  • 科学上网

挂载盘：挂载盘客户端让你可以通过本地虚拟磁盘直接访问服务器上的文件，无需同步。不占用本地磁盘。也就是说，在“我的电脑”里，会直接多出一个盘。

Seafile的网页版有历史版本功能，该功能十分强大。
