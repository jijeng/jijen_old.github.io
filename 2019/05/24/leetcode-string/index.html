  1. longest substring without repeating characters

一开始还以为使用dp，但是使用双指针+ dictionary 就可以解决。求解的是子串（不是子序列），所以一定是一个连续的区间 [i, j]，有点贪心的思想，通用的“滑动窗口 + 哈希表” 的做法。由于 $i$, $j$ 最多增加 $n$ 次，且hash 表的插入和更新时间复杂度是$O(1)$， 所以总的时间复杂度是$O(n)$。
