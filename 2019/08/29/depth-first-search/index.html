Depth-first Search

Posted on 2019-08-29 | Post modified: 2019-08-31 | In 算法 | |

深度优先搜索专题，题目来源于LeetCode。大多数题目使用c++ 实现，少量使用python 实现。

使用场景：当状态数量非常庞大的时候，解的个数非常少的时候，适合使用深度优先搜索。
但搜索不等于深度优先搜索。搜索是一种思想，深搜索是一种实现方式。
