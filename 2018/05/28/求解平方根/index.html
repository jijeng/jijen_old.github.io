尾递归

尾调用定义:
在某个函数的最后一步调用另一个函数.

上面代码中，函数m和n都属于尾调用，因为它们都是函数f的最后一步操作。不一定是最后一行。

同理是可以推广到尾递归。

尾递归的意义在于防止栈溢出。因为递归的调用是非常消耗内存的，需要再栈中保存N 个调调用记录，很容易发生栈溢出 (stack overflow) 。但是对于为递归来说，只有一个调用记录，因为上一个已经结束，所以是不会发生 “栈溢出”错误的。

子数组的最大乘积

给定一个长度为 N 的整数数组，只允许用乘法，不能用除法， 计算任意 ( N-1) 个数的组合中乘积最大的一组。

第一种解法：

暴力求解，遍历出n 个，然后需要 n-1 次相乘，所以时间复杂度是 $O(N^2)$

···c++

#include

using namespace std;

class Solution {
public:
int maxProduct(int *nums) {

    int res;
    int max;
    int i, j;
    int n= sizeof(nums) /sizeof(nums[0]);

    for(i =0; i<n; i++)
    {
        for(j =0; j<n; j++)
        {
            res =1;
            res *= ( (i==j) ? 1: nums[j]); // 第二个for 循环说明的是 n-1 个数字都是要给乘起来的
            if (res ==0) break;
        }
        max =( i==0 ? res: max);
        max =( max < res? res: max);
    }
    return max;

}

};
